# Notes

Time Complexity: O(n)
Space Complexity: O(n)

Why this matters for protocols:

- Hash maps are fundamental in:
  - transaction indexing
  - state storage
  - mempool lookups

- Similar patterns appear in:
  - UTXO tracking (Bitcoin)
  - account state mapping (Ethereum)

Insight:
Efficient lookup structures are critical for performance in high-throughput systems.
